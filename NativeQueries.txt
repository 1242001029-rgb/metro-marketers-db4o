NATIVE QUERIES - Metro Marketers Customer Marketing Database

NATIVE QUERY 1
Goal: Select all customers.
Query:
db.query(new Predicate<Customer>() {
    public boolean match(Customer c) {
        return true;
    }
});
Expected Result: Display all customer objects.

NATIVE QUERY 2
Goal: Select customers from Jakarta.
Query:
db.query(new Predicate<Customer>() {
    public boolean match(Customer c) {
        return c.toString().contains("Jakarta");
    }
});
Expected Result: Display customers whose city is Jakarta.

NATIVE QUERY 11
Goal: Sort customers by name.
Query:
List<Customer> sortedCustomers = new ArrayList<Customer>(
    db.query(new Predicate<Customer>() {
        public boolean match(Customer c) {
            return true;
        }
    })
);
sortedCustomers.sort(new Comparator<Customer>() {
    public int compare(Customer c1, Customer c2) {
        return c1.getName().compareToIgnoreCase(c2.getName());
    }
});
Expected Result: Display customer list sorted alphabetically by name.
