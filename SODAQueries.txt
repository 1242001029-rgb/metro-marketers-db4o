SODA QUERIES - Metro Marketers Customer Marketing Database

SODA QUERY 1
Goal: Select all customers.
Query:
Query q1 = db.query();
q1.constrain(Customer.class);
q1.execute();
Expected Result: Display all customers.

SODA QUERY 2
Goal: Search customer by name Alice Tan.
Query:
Query q2 = db.query();
q2.constrain(Customer.class);
q2.descend("name").constrain("Alice Tan");
q2.execute();
Expected Result: Display customer data for Alice Tan.

SODA QUERY 10
Goal: Search customers from Bekasi or Bandung.
Query:
Query q10 = db.query();
